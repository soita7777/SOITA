# Blockchain Secure Voting System

A decentralized voting system using blockchain technology to fully secure the voter's data and ensure transparency.

## Features
- Secure voting
- One person one vote
- Transparent results
- Blockchain based technology

## Tech Stack
- React
- Node.js
- Solidity(I prefferd v. 8.0.11)
- Ethereum

## Run Project
npm install
npm start
